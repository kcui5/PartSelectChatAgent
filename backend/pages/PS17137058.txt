CONFIGURED UI BOARD Specifications
This is the user interface control board for various dishwashers. The user interface panel is what you directly interact with on your dishwasher to start and stop it and select the specific cycle you want. If your user interface panel is not responding and you have tested your dishwasher to ensure no other part could be at fault, it will need to be replaced immediately for it to regain function. Replacing this panel will requires some disassembly of your dishwasher, and will require tools like a screwdriver, wrench, and a pair of pliers. This OEM user interface panel measures approximately 14-inches wide and is sold individually.
This part fixes the following symptoms: Will Not Start
